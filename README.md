# CPU Scheduling Analysis Tool

# Description
A GUI-based CPU Scheduling Analysis Tool developed using Python. This application simulates and compares different CPU scheduling algorithms with performance metrics and visualizations.

# Features
- Implemented scheduling algorithms:
  - First Come First Serve (FCFS)
  - Shortest Job First (SJF)
  - Round Robin (RR)
  - Priority Scheduling
- Calculates:
  - Completion Time (CT)
  - Turnaround Time (TAT)
  - Waiting Time (WT)
- Gantt Chart visualization
- Algorithm comparison
- Graphical performance analysis
- Interactive GUI using Tkinter

# Technologies Used
- Python
- Tkinter (GUI)
- Matplotlib (Visualization)
